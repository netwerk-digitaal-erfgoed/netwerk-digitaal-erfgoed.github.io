# Dataplatforms

[Collectie-informatiesystemen](/glossary.md#collection-management-system) moeten de volgende API's implementeren:

- [Termennetwerk-API](/services/network-of-terms/index.md) om brondata te beschrijven met [termen](/glossary.md#term)
- [Datasetregister-API](/services/dataset-register/index.md) om gepubliceerde datasets te registreren.

Raadpleeg voor meer informatie de [Requirements voor collectie-informatiesystemen](https://docs.nde.nl/requirements-collection-management-systems/).
