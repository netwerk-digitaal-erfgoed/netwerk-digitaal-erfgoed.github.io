# Reconciliation API

Gebruik de [Reconciliation Service API](https://reconciliation-api.github.io/specs/latest/)
om **tekstuele waarden in je data te matchen en te reconciliëren met [termen](../../glossary.md#term)**.


Als je de Reconciliation Service API wilt gebruiken, configureer dan één of meer service-endpoints in je
[OpenRefine](https://openrefine.org)-applicatie. De URL van een endpoint heeft de structuur
`https://termennetwerk-api.netwerkdigitaalerfgoed.nl/reconcile/{sourceUri}`. Een volledige lijst van endpoints vind je op
de [Termennetwerk-website](https://termennetwerk.netwerkdigitaalerfgoed.nl/reconciliation).

Let op: het Termennetwerk biedt Reconciliation-endpoints alleen aan voor terminologiebronnen die zelf geen
dergelijke endpoints aanbieden.
