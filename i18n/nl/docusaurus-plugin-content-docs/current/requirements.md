# Requirements

NDE publiceert requirements om de interoperabiliteit in het Nederlandse erfgoednetwerk te verbeteren:

* [Requirements voor datasets](https://docs.nde.nl/requirements-datasets/) beschrijven hoe [datasetbeschrijvingen](glossary.md#dataset-description) gepubliceerd moeten worden
* [Schema.org Application Profile](https://docs.nde.nl/schema-profile/)
* [Requirements voor collectie-informastiesystemen](https://docs.nde.nl/requirements-collection-management-systems/) beschrijven de vereisten waaraan [collectie-informastiesystemen](glossary.md#collection-management-system) moeten voldoen om NDE-compatibel te worden
* [Requirements voor Termennetwerk-implementaties](https://docs.nde.nl/requirements-network-of-terms-implementations/) beschrijven hoe  collectie-informastiesystemen de Termennetwerk-API moeten implementeren
* [Requirements voor terminologiebronnen](https://docs.nde.nl/requirements-terminologiebronnen/) beschrijven de functionele en niet-functionele vereisten waaraan [terminologiebronnen](glossary.md#terminology-source) moeten voldoen om opgenomen te worden in het [Termennetwerk](glossary.md#network-of-terms)
